Отчет к домашнему заданию №3

1. Условие задачи

В рамках выполнения данного дз требуется разработать программу строкового калькулятора. Причём по запросу с клавиатуры в консольном приложении вводится строка, которая может содержать:

-знаки операций +,-,*,/;
-константы (целые или вещественные);
-скобки ();
-функции sin, cos, tg, ctg, exp;
-переменную x.
-Если переменная x есть в формуле, то запрашивается её значение, после чего вычисляется результат выражения и выводится на экран.

2. Схема алгоритма

Программа строкового калькулятора работает по следующему алгоритму:

запись исходной формулы в постфиксной форме;
определение наличия переменной x в данной формуле и запрос её значения;
вычисление значения данного выражения при помощи его записи в постфиксной форме;
вывод значения на экран пользователя.

3. Контрольный расчёт

На вход программе подается следующее выражение: cos(x-5)*10/(2*x-5)
Программа запрашивает значение переменной x.
Вводится значение 5.
Программа выводит результат "2", что является верным ответом.

4. Вывод

В результате выполнения данного домашнего задания была изучена обратная польская нотация, создан рабочий строковый калькулятор, полностью соответствующий условию задачи.
